HKBusFind

This application provide users to find related public transport route informations, including route stops, timetables, and arrival time.

Functions

About HKBusFind

This App provide these hong Kong Public Transport route infos:

  • Kowloon Motor Bus
  • New World First Bus
  • Citybus
  • New Lantau Bus
  • MTR (LRT) Shuttle Bus
  • Green Minibus（Only available for the route which have provided ETA Service)
  • Hong Kone Tramway

This App provided these functions：

  • Find Nearby / related location on the Map’s routes.
  • Different route on the same stop.
  • Related stop / route stop’s estimate time arrival (ETA).
  • Bookmark Stops / Different Scene of stop’s route.
  • Support Apple Watch display selected scene of bookmarked stop route’s ETA.

Future plan will provide：

  • Hong Kong Ferry’s route information.



Estimated Time of Arrival (ETA) information is provided by data.gov.hk . This application does not guarantee its accuracy.

Nearby Location

1. When open the app, if the update view occur. Let's wait for download the route data complete.

2. If the app asked for the location access right, please accept it for use the full function of the app.

3. Then the app will show the current location of the stops in map and routes in table nearby you.

4. As you can see the "<" button in the map area, press it and expand the menu buttons set.

As shown on the above screen's buttons:

a. A button to collapse the menu buttons set.

b. Show Search Route Function.

c. Scan nearby location's stops and routes.

d. Scan the current map location's stops and routes. You can move the map and scan related location.

e. Show bookmarked stops.

f. Show bookmarked scene's stop route set.

g. Show apps Settings.

Search Route

1. On this view, you can search the route number you want and show the result on the table.

2. You can see the related route's information when you press related route's table cell.

a. Different route color present at least one company to provide services:

38 - Kowloon Motor Bus

38 - Citybus

38 - New World First Bus

38 - New Lantau Bus

38 - Green Minibus

3 - Hong Kone Tramway

K52 - MTR (LRT) Shuttle Bus

b. If you press this button will go back to "Nearby" location view.

Route informations

1. On this view, you can see related route's informations.

2. Such as route stop, time table and BBI informations. (as shown above)

a. This button will show related action sheet with different functions (as shown below)

b. If you press this button will go back to "Nearby" location view.

Bookmark Stops

1. On this view, you can see the bookmarked stops list.

2. You can directly tap related stop table cell to see related routes in this stop.

3. Also you can slide to right to remove bookmarked stop (as shown below)

Bookmark Stop Route

1. On this view, you can see the bookmarked stop route scene list.

2. Slide on left and tap on related bookmarked stop route scene who can delete related scene include ALL related bookmarked stop route.

a. Add scene button which will popup the add scene alert, provide scene name and create a new scene. (As shown below)

b. Select related "Watch" button will sync selected bookmark stop route informations to watch and show these route's ETA records. (As shown below)

3. Slide on right and tap on related bookmarked stop route scene which will go to edit bookmarked stop route view.

4. Slide on left and tap on related bookmarked stop route who can delete related bookmarked stop route in this Scene.

5. On upper right button, tap it and popup the change scene name alert and let you change it.

Setting

1. On this view, you can see the app settings.

2. You can change language (as shown below), remove ALL bookmarks, view this support page and app version number.
